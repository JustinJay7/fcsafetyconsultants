# FC Safetey Consultants — Modern Responsive Website

**A clean, elegant, and high-performance website built with modern web technologies.**

## 🌟 Overview

The design emphasizes minimalism, sophisticated typography, and subtle animations that create an immersive and memorable feel without sacrificing performance.

Whether used as a portfolio, agency landing page, SaaS product showcase, or creative studio site, Aether delivers a professional and polished presence across all devices.

## ✨ Key Features

- **Fully Responsive** — Optimized for desktop, tablet, and mobile with a mobile-first approach
- **Smooth Animations** — Elegant scroll-triggered effects, hover interactions, and page transitions
- **Fast Performance** — Optimized assets and clean code for lightning-fast load times
- **Modern UI/UX** — Clean layout, beautiful typography, and intuitive navigation
- **Accessibility Focused** — Semantic HTML and ARIA support
- **SEO Optimized** — Proper meta tags, structured content, and fast Core Web Vitals
- **Dark/Light Mode Ready** — Built with theme flexibility in mind

## 🎨 Design Philosophy

Aether follows a minimalist yet luxurious design language. The focus is on whitespace, elegant typography, and subtle micro-interactions that delight users while maintaining clarity and purpose. Every visual element serves the overall experience.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router) + React 19
- **Styling**: Tailwind CSS + custom CSS
- **Animations**: Framer Motion + GSAP
- **Icons**: Lucide React
- **Deployment**: Vercel
- **Other**: TypeScript, ESLint, Prettier
